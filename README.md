# MorphingOscillatorDemo
Morphing waveform oscillator for JUCE synthesizer

This repo contains a reimplementation of JUCE's AudioSynthesiserDemo example project with a morphing waveform oscillator. The waveform is controlled by a single knob, allowing the user to fade between sine, square, and triangle waves.

## Compiling and running the demo
Open the .projucer project file in JUCE's Projucer and export to your chosen operating system.

Tested on macOS (Sequoia 15.7.4 on Intel and Apple Silicon), Windows 11 (amd64), Ubuntu 22.02 (amd64), and Raspberry Pi OS (Raspberry Pi 5 with ARMv8 processor).
